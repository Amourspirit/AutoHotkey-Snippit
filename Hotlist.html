Hotlist Graphic User Interface


The Hotlist is where you can view, swap, filter and edit exiting profiles and plugin. Most all changes will take place from the Hotlist interface.

Exit Closes the Hotlist


Filters:

Profile Filter allows you to view the other available profiles


Plugin Filter will filter the list to show only items from the selected plugin


Type Filter will filter the list to show only the selected type such as Hotkey or Hotstring.


Category Filter will filter the list to show only items in the user defined category


Save, When item in the list are changed such as disabled or when a profile state is changed (enable or disabled) the list must be save before the changes will take effect.

Note: When other edits are done such as New Profile, Edit Profile, New Plugin or Edit plugin the changes will take place immediately.
